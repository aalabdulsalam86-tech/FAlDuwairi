## 🎮 تجربة الطالب

1. يدخل الصفحة الرئيسية → يختار الصف
2. يختار المادة
3. يختار الفصل الدراسي (الأول أو الثاني)
4. **تبدأ اللعبة مباشرة** — بدون خطوات إضافية

### في كل لعبة جديدة:
- ✅ **١٠ أسئلة عشوائية** من المجموعة الكاملة للمادة
- ✅ **ترتيب الخيارات يُخلط** في كل سؤال
- ✅ **لون الواجهة** يتغير عشوائياً (١٠ ألوان)
- ✅ **خلفية الحركة** تتغير عشوائياً (٤ أنماط)
- ✅ **شرح** يظهر بعد كل إجابة
- ✅ في النهاية: **النقاط + مراجعة كاملة** لجميع الإجابات

## ✏️ تعديل الأسئلة أو إضافة المزيد

كل مادة/فصل لها ملف JSON منفصل في مجلد `games/`. مثال: `bio12_s1.json` لأحياء الثاني عشر علمي فصل ١.

افتح الملف في محرر نصوص وعدّل:
```json
{
  "type": "mcq",
  "q": "نص السؤال؟",
  "opts": ["خيار 1", "خيار 2", "خيار 3", "خيار 4"],
  "ans": 0,
  "exp": "الشرح بعد الإجابة."
}
```

- `ans`: رقم الخيار الصحيح (**يبدأ من 0**)
- `exp`: الشرح الذي يظهر للطالب بعد الإجابة
- يمكنك إضافة أي عدد من الأسئلة الجديدة

## 📝 ملاحظات مهمة

### ⚠️ مواد تحتاج أسئلة إضافية
بعض مواد الصف الثاني عشر علمي بها ١٠ أسئلة فقط — النظام يعرض ١٠ لكل لعبة، لذا سترى نفس الأسئلة. يُنصح بإضافة المزيد.

### 🎯 الشروحات
- **الأحياء الصف العاشر (فصل ١ و ٢):** شروحات مفصّلة مكتوبة يدوياً
- **باقي المواد:** شروحات تلقائية قصيرة

### 🔧 البنية التقنية
- **ملف واحد للمحرك:** `games/game.js` يدير جميع الألعاب
- **ملفات JSON منفصلة:** يسهل التعديل
- **index.html واحد:** يوجّه الطالب تلقائياً
